# UPC Quant Society website

The institutional website for UPC Quant Society, built as a static site with Astro 7 and TypeScript.

## Languages and routes

The website is available in Catalan and English:

- Catalan pages use the `/ca/` route prefix.
- English pages use the `/en/` route prefix.
- The root route redirects to `/ca/`.

The two localized home routes are separate Astro pages that render the shared `src/components/HomePage.astro` component. Other localized pages are generated by `src/pages/[lang]/[slug].astro`.

## Content

`src/data/site.ts` is the central content source. It contains:

- Catalan and English page content.
- Navigation links and interface labels.
- Localized route pairings used by the language switch and `hreflang` metadata.
- Shared contact and external-link information.

Keep the Catalan and English versions synchronized when changing content or routes.

## Local development

The project requires Node.js 22.12.0 or newer.

Install dependencies:

```sh
npm install
```

Start the local development server:

```sh
npm run dev
```

Astro serves the site at `http://localhost:4321` by default.

## Production build

Generate the static production site in `dist/`:

```sh
npm run build
```

Preview the generated site locally:

```sh
npm run preview
```

The generated `dist/` directory, Astro type output, and installed dependencies are ignored by Git.

## GitHub Pages deployment

The site is configured for `https://upc-quant-society.github.io` in `astro.config.mjs`.

The workflow in `.github/workflows/deploy.yml` builds and deploys the static site to GitHub Pages on pushes to `main`. It can also be started manually through GitHub Actions with `workflow_dispatch`.
